Intelligent E-Learning Platform: Simplified Flow

1. The Learner's Input (The Request)
Visually Impaired Learner
Uses Voice Command to ask a question or navigate.
Web Interface (PHP/HTML) captures the input.
2. The Core Processing (The Brain)
PHP Web Server (Logic Controller)
MySQL Database: Retrieves student's Progress & Performance for personalization.
External AI Model API: Receives the question, and the NLP Conversational Tutor generates a personalized, tailored answer.
3. Accessible Output (The Answer)
The AI's Answer is immediately routed to two services:
Text-to-Speech (TTS) Module: Converts text to AUDIO OUTPUT.
Braille Translation Module: Converts text to BRAILLE OUTPUT.
Delivered to the Learner's Device / Assistive Hardware.
